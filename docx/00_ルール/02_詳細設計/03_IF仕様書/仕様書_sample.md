# IF-XX-XX 注文データCSV出力

## 基本情報

| 項目 | 内容 |
| :--- | :--- |
| **IF名** | 注文データCSV出力 |
| **出力形式** | CSV（カンマ区切り、ダブルクォーテーション囲みあり、ヘッダー行あり） |
| **文字コード** | `UTF-8` |
| **実行タイミング** | 管理画面の「CSV出力」ボタン押下時 |

---

## 目的

> 他システム連携および分析用として、**確定済みの注文データ** をCSV形式で出力する。

---

## 対象データ

- **期間**：指定された期間内
- **ステータス**：`完了` または `配送中`

---

## 機能仕様

### ■ 項目一覧

| No | 項目名 | 物理名 | 型 | 繰り返し |
| :--- | :--- | :--- | :--- | :--- |
| 1 | 注文ID | `order_id` | `string` | - |
| 2 | 注文日時 | `order_datetime` | `string` | - |
| 3 | 顧客名 | `customer_name` | `string` | - |
| 4 | 合計金額 | `total_amount` | `number` | - |
| 5 | 商品名（明細） | `item_name` | `string` | 1 |
| 6 | 数量 | `quantity` | `number` | 1 |

#### ▶ 備考

| 項目名 | 備考 |
| :--- | :--- |
| 商品名（明細） | **1注文で複数行** 出力される |

### ■ 書式（Excel / PDF 出力時のみ）

> ※ CSV出力の場合は本セクションは不要。Excel / PDFで出力する場合のみ、以下のような書式情報を記載する。

| 項目名 | フォント | 文字サイズ | 太字 | 文字色 | 背景色 | セル幅 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 注文ID | `Meiryo UI` | `10pt` | - | 黒 | - | 15 |
| 注文日時 | `Meiryo UI` | `10pt` | - | 黒 | - | 20 |
| 合計金額 | `Meiryo UI` | `10pt` | ✅ | 黒 | 薄黄 | 12 |

#### ▶ 備考

| 項目名 | 備考 |
| :--- | :--- |
| 合計金額 | 右寄せ・カンマ区切り |

### ■ 変換・抽出ロジック

| 項目名 | ロジック |
| :--- | :--- |
| 注文ID | そのまま出力 |
| 注文日時 | `YYYY/MM/DD HH:mm:ss` 形式に変換 |
| 顧客名 | `orders.customer_id` で `customers` をJOINして取得 |
| 合計金額 | **カンマ区切りなし** の数値として出力 |
| 商品名（明細） | 注文に紐づく明細行を **繰り返し出力** |
| 数量 | そのまま出力 |

---

## 参照テーブル

### ■ `orders`

| 項目名 | カラム | データ型 |
| :--- | :--- | :--- |
| 注文ID | `id` | `string` |
| 注文日時 | `created_at` | `string` |
| 合計金額 | `amount` | `number` |

### ■ `customers`

| 項目名 | カラム | データ型 |
| :--- | :--- | :--- |
| 顧客名 | `name` | `string` |

### ■ `order_items`

| 項目名 | カラム | データ型 |
| :--- | :--- | :--- |
| 商品名（明細） | `item_name` | `string` |
| 数量 | `quantity` | `number` |

---

## 備考

- データが **0件** の場合は、**ヘッダー行のみ** のファイルを出力する。
- ファイル名は `OrderData_YYYYMMDDHHMMSS.csv` とする。
