# 06-01-01 ロードマップアイテム登録画面

## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | 06-01-01 |
| 画面名 | ロードマップアイテム登録画面 |
| URL | `/projects/{project_id}/roadmap/new` |

---

## 目的

ロードマップアイテムを新規作成する。

---

## 処理概要

### ■ 画面の遷移

| 種別 | 画面 |
|------|------|
| 遷移元 | 06-01-00 プロダクトロードマップ画面 |
| 遷移先（保存） | 06-01-00 プロダクトロードマップ画面 |
| 遷移先（キャンセル） | 06-01-00 プロダクトロードマップ画面 |

---

## アクセス権限

| 種別 | 内容 |
|------|------|
| アクセス権限 | admin 以上 |

---

## 機能仕様

### ■ 入力フィールド

| 項目名 | 必須 | 型 | 初期値 | データ型 |
|--------|------|----|--------|---------|
| タイトル | ✅ | テキストボックス | 空 | string |
| 説明 | - | テキストエリア | 空 | string |
| クォーター | ✅ | セレクトボックス | 空 | UUID |
| ステータス | ✅ | セレクトボックス（計画中/進行中/完了/保留） | 計画中 | string |

### ■ バリデーション

| 項目名 | 条件 |
|--------|------|
| タイトル | 必須 |
| クォーター | 必須・同プロジェクト内のクォーターであること |
| ステータス | 計画中 / 進行中 / 完了 / 保留 のいずれか |

### ■ ボタン

| ボタン名 | 挙動 |
|---------|------|
| 保存 | `POST /api/v1/projects/{project_id}/roadmap/` → ロードマップ画面へ遷移 |
| キャンセル | ロードマップ画面へ戻る（変更なし） |

---

## アクション

### ■ 画面遷移時

- API: `GET /api/v1/projects/{project_id}/quarters/` でクォーター選択肢を取得

---

## 状態管理

| 状態 | 内容 |
|---|---|
| form | 入力フォームの値（title, description, quarter_id, status） |
| quarters | クォーター選択肢 |
| isSaving | 保存処理中は `true` でボタン非活性 |
