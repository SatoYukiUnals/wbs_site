# 【画面仕様書】プロジェクト別ダッシュボード画面

## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 07-01-00 |
| 画面名 | プロジェクト別ダッシュボード画面 |
| URL | `/projects/{id}/dashboard` |
| 対応デザイン | - |

## 2. 目的

特定プロジェクトの進捗・ステータス・担当タスクを詳細に可視化する。

## 3. 処理概要

### 画面で操作できる情報

- プロジェクト全体の進捗サマリー
- タスクのステータス別件数
- メンバー別担当タスク数
- 期限切れ・直近期限タスクの一覧

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| プロジェクト詳細画面 | ダッシュボードリンク | プロジェクト別ダッシュボード画面 |
| プロジェクト別ダッシュボード画面 | タスク名クリック | タスク詳細画面 |

## 8. アクション

### 画面表示時

1. `GET /api/v1/projects/{id}/dashboard/` 呼び出し
2. プロジェクトダッシュボードデータを表示
