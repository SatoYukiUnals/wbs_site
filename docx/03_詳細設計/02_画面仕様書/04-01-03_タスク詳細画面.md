# 04-01-03 タスク詳細画面

## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | 04-01-03 |
| 画面名 | タスク詳細画面 |
| URL | `/projects/{project_id}/tasks/{task_id}` |

---

## 目的

タスクの全情報を表示・編集する。権限に応じて編集可能な項目が異なる。

---

## 処理概要

### ■ 画面の遷移

| 種別 | 画面 |
|------|------|
| 遷移元 | 04-01-00 WBS一覧 |

---

## アクセス権限

| 種別 | 内容 |
|------|------|
| アクセス権限 | プロジェクトメンバー以上 |

---

## ダイアログ

なし

---

## 機能仕様

### ■ 入力フィールド（admin以上のみ編集可）

| 項目名 | 必須 | 型 | 初期値 | データ型 |
|--------|------|----|--------|---------|
| タスク名 | ✅ | テキストボックス | 現在値 | string |
| 説明 | - | テキストエリア | 現在値 | string |
| 開始日 | - | 日付 | 現在値 | date |
| 終了日 | - | 日付 | 現在値 | date |
| 工数（時間） | - | 数値 | 現在値 | number |
| 優先度 | ✅ | セレクトボックス（高/中/低） | 現在値 | string |
| クォーター | - | セレクトボックス | 現在値 | UUID |
| 担当者 | - | 複数選択 | 現在値 | UUID[] |

### ■ 入力フィールド（admin以上のみ編集可・task_type='task' の場合のみ表示）

| 項目名 | 必須 | 型 | 初期値 | データ型 | 選択肢 |
|--------|------|----|--------|---------|--------|
| タスク種別（task_kind） | - | セレクトボックス | 現在値 | string | 未設定 / 実装 / ドキュメント作成 / レビュー依頼 / レビュー修正 |

### ■ 入力フィールド（担当者 or admin 編集可）

| 項目名 | 必須 | 型 | 初期値 | データ型 | 選択肢 |
|--------|------|----|--------|---------|--------|
| ステータス | ✅ | セレクトボックス | 現在値 | string | Todo / InProgress / InReview / Done / OnHold |

### ■ ボタン

| ボタン名 | 挙動 |
|---------|------|
| 保存（admin以上） | `PUT /api/v1/projects/{project_id}/tasks/{task_id}/` |
| ステータス更新 | `PATCH /api/v1/projects/{project_id}/tasks/{task_id}/` |
| テンプレート適用（admin以上） | テンプレート選択ダイアログ → `POST apply-template` |

---

## アクション

### ■ 画面遷移時

- API: `GET /api/v1/projects/{project_id}/tasks/{task_id}/`
