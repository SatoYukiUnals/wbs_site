## 1. 画面ID体系

```
[カテゴリ番号]-[データ単位番号]-[操作区分]
```

| 要素 | 桁数 | 説明 |
|---|---|---|
| カテゴリ番号 | 2桁 | 機能カテゴリ（下表参照） |
| データ単位番号 | 2桁 | カテゴリ内のデータ単位（同エンティティは同番号） |
| 操作区分 | 2桁 | 00=一覧 / 01=登録 / 02=編集 / 03=詳細 / 04以降=その他 |

### カテゴリ番号定義

| カテゴリ番号 | カテゴリ名 |
|---|---|
| 01 | 認証・ユーザー管理 |
| 02 | プロジェクト管理 |
| 03 | クォーター管理 |
| 04 | WBS・タスク管理 |
| 05 | ガントチャート ※WBS画面に統合 |
| 06 | プロダクトロードマップ |
| 07 | ダッシュボード |
| 08 | レビュー管理 |
| 09 | 報告書管理 |
| 10 | Excel出力 |
| 11 | テンプレート管理 |

### 05 ガントチャート ※WBS画面に統合

> **統合**：ガントチャートは独立した別画面としては実装しない。04-01-00 WBS一覧画面の右側カラムとしてインライン表示する。
