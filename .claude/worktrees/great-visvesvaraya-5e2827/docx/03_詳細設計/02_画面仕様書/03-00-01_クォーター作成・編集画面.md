# 【画面仕様書】クォーター作成・編集画面

## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 03-00-01 |
| 画面名 | クォーター作成・編集画面 |
| URL | `/projects/{id}/quarters/new`（作成） / `/projects/{id}/quarters/{quarter_id}/edit`（編集） |
| 対応デザイン | - |

## 2. 目的

クォーター（四半期期間）の新規作成または既存クォーターの編集を行う。

## 3. 処理概要

### 画面で操作できる情報

- クォーター名・開始日・終了日の入力
- クォーターの保存

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| クォーター一覧画面 | 新規作成・編集ボタン | クォーター作成・編集画面 |
| クォーター作成・編集画面 | 保存成功 | クォーター一覧画面 |
| クォーター作成・編集画面 | キャンセルボタン | クォーター一覧画面 |

## 4. アクセス権限

| ロール | アクセス可否 |
| :--- | :--- |
| master | ○ |
| admin（プロジェクト内） | ○ |
| member | - |
| 未ログイン | - |

## 7. バリデーション

| 項目名 | トリガー | チェック内容 | エラーID | エラーメッセージ |
| :--- | :--- | :--- | :--- | :--- |
| クォーター名 | submit | 必須・1〜100文字 | ERR_VAL_001 | クォーター名を入力してください |
| 開始日 | submit | 必須・日付形式 | ERR_VAL_001 | 開始日を入力してください |
| 終了日 | submit | 必須・開始日以降 | ERR_VAL_001 | 終了日は開始日以降の日付を入力してください |

## 8. アクション

### 画面表示時（編集モード）

1. `GET /api/v1/projects/{id}/quarters/{quarter_id}/` 呼び出し
2. 取得データをフォームに設定

### 保存ボタン押下時

1. フロント側バリデーション実行
2. API呼び出し（POST or PUT）
3. 成功: `/projects/{id}/quarters` へ遷移
