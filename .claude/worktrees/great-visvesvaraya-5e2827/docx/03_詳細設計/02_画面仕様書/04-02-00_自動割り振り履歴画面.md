## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 04-02-00 |
| 画面名 | 自動割り振り履歴画面 |

| 対応デザイン | - |

## 2. 目的

自動割り振り（担当者自動割り当て）の実行履歴を確認する。admin以上のみ参照可能。

## 3. 処理概要

### 画面で操作できる情報

- 自動割り振り実行履歴の一覧表示
- 各実行ログの詳細参照

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| WBS一覧画面 | 自動割り振り履歴リンク | 自動割り振り履歴画面 |
| 自動割り振り履歴画面 | 詳細ボタン | 自動割り振り履歴詳細（同画面内スライドパネル or 遷移） |
