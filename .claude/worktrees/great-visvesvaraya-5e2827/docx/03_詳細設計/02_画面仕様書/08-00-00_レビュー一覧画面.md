# 【画面仕様書】レビュー一覧画面

## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 08-00-00 |
| 画面名 | レビュー一覧画面 |
| URL | `/projects/{id}/reviews` |
| 対応デザイン | - |

## 2. 目的

プロジェクト内のレビュー申請一覧を表示し、レビュー実施・詳細参照を行う。

## 3. 処理概要

### 画面で操作できる情報

- レビュー一覧の表示（ステータスフィルタ付き）
- レビュー実施画面への遷移
- レビュー詳細・コメント画面への遷移

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| プロジェクト詳細画面 | レビューリンク | レビュー一覧画面 |
| レビュー一覧画面 | レビューボタン | レビュー実施画面 |
| レビュー一覧画面 | 詳細ボタン | レビュー詳細・コメント画面 |

## 4. アクセス権限

| :--- | :--- |

## 8. アクション

### 画面表示時

1. `GET /api/v1/projects/{id}/reviews/` 呼び出し
2. レビュー一覧を表示

### ステータスフィルタ変更時

1. フィルタパラメータを付けて `GET /api/v1/projects/{id}/reviews/?status={status}` 呼び出し
