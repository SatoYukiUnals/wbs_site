# 【画面仕様書】クォーター一覧画面

## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 03-00-00 |
| 画面名 | クォーター一覧画面 |
| URL | `/projects/{id}/quarters` |
| 対応デザイン | - |

## 2. 目的

プロジェクトに紐づくクォーター（四半期期間）の一覧を表示し、クォーターの作成・編集・削除を行う。

## 3. 処理概要

### 画面で操作できる情報

- クォーター一覧の表示
- クォーターの作成・編集・削除（admin以上）

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| プロジェクト詳細画面 | クォーター管理リンク | クォーター一覧画面 |
| クォーター一覧画面 | 新規作成ボタン | クォーター作成・編集画面 |
| クォーター一覧画面 | 編集ボタン | クォーター作成・編集画面 |

## 4. アクセス権限

| ロール | アクセス可否 |
| :--- | :--- |
| メンバー以上（プロジェクトメンバー） | ○ |
| 非メンバー | - |
| 未ログイン | - |

## 5. ダイアログ

| ダイアログ名 | 表示条件 | 内容 |
| :--- | :--- | :--- |
| 削除確認ダイアログ | 削除ボタン押下 | 「このクォーターを削除しますか？」確認・キャンセルボタン |

## 7. バリデーション

なし

## 8. アクション

### 画面表示時

1. `GET /api/v1/projects/{id}/quarters/` 呼び出し
2. クォーター一覧を表示

### 削除確認「削除」押下時

1. `DELETE /api/v1/projects/{id}/quarters/{quarter_id}/` 呼び出し
2. 成功（204）: 一覧を再取得して表示
