# 【画面仕様書】タスク一括作成画面

## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 04-01-04 |
| 画面名 | タスク一括作成画面 |
| URL | `/projects/{id}/tasks/bulk` |
| 対応デザイン | - |

## 2. 目的

テンプレートまたは自由入力によりタスクを一括作成する。admin以上のみ操作可能。

## 3. 処理概要

### 画面で操作できる情報

- テンプレートの選択（オプション）
- タスク名リストの入力（改行区切り）
- 親タスク・開始日・終了日の一括設定
- 複数タスクの一括作成

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| WBS一覧画面 | 一括作成ボタン | タスク一括作成画面 |
| タスク一括作成画面 | 作成成功 | WBS一覧画面 |
| タスク一括作成画面 | キャンセルボタン | WBS一覧画面 |

## 7. バリデーション

| 項目名 | トリガー | チェック内容 | エラーID | エラーメッセージ |

## 8. アクション

### 画面表示時

1. `GET /api/v1/templates/?type=task` で利用可能なテンプレートを取得
2. `GET /api/v1/projects/{id}/tasks/` で親タスク候補を取得

### テンプレート選択時

1. 選択されたテンプレートの `content` をタスクリストエリアに反映

### 作成ボタン押下時

1. フロント側バリデーション実行
2. `POST /api/v1/projects/{id}/tasks/bulk/` 呼び出し
3. 成功（201）: `/projects/{id}/wbs` へ遷移
4. 失敗（400）: エラーメッセージを表示
