# 【画面仕様書】ロードマップ画面

## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 06-00-00 |
| 画面名 | ロードマップ画面 |
| URL | `/projects/{id}/roadmap` |
| 対応デザイン | - |

## 2. 目的

プロジェクトのロードマップアイテム（マイルストーン・フェーズ等）を時系列で可視化する。

## 3. 処理概要

### 画面で操作できる情報

- ロードマップアイテムの一覧・タイムライン表示
- アイテムの作成・編集・削除（admin以上）

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| プロジェクト詳細画面 | ロードマップリンク | ロードマップ画面 |
| ロードマップ画面 | 新規作成ボタン | ロードマップアイテム作成・編集画面 |
| ロードマップ画面 | 編集ボタン | ロードマップアイテム作成・編集画面 |

## 4. アクセス権限

| ロール | アクセス可否 |
| :--- | :--- |
| メンバー以上（プロジェクトメンバー） | ○ |
| 非メンバー | - |
| 未ログイン | - |

## 5. ダイアログ

| ダイアログ名 | 表示条件 | 内容 |
| :--- | :--- | :--- |
| 削除確認ダイアログ | 削除ボタン押下 | 「このアイテムを削除しますか？」確認・キャンセルボタン |

## 7. バリデーション

なし

## 8. アクション

### 画面表示時

1. `GET /api/v1/projects/{id}/roadmap/` 呼び出し
2. ロードマップアイテムをタイムラインで表示

### 削除確認「削除」押下時

1. `DELETE /api/v1/projects/{id}/roadmap/{item_id}/` 呼び出し
2. 成功（204）: 一覧を再取得して表示

## 9. 状態管理

| 状態名 | 型 | 初期値 | 説明 |
| :--- | :--- | :--- | :--- |

| isLoading | boolean | false | 読み込み中フラグ |
