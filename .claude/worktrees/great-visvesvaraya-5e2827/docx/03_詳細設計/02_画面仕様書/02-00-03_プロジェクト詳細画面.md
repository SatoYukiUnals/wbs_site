# 【画面仕様書】プロジェクト詳細画面

## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 02-00-03 |
| 画面名 | プロジェクト詳細画面 |
| URL | `/projects/{id}` |
| 対応デザイン | - |

## 2. 目的

プロジェクトの基本情報（名称・説明・期間・ステータス）を表示し、各機能への入口を提供する。

## 3. 処理概要

### 画面で操作できる情報

- プロジェクト基本情報の参照
- プロジェクト編集・削除（admin以上）
- 各機能（WBS・ガントチャート・ロードマップ等）への遷移

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| プロジェクト一覧画面 | プロジェクト行クリック | プロジェクト詳細画面 |
| プロジェクト詳細画面 | 編集ボタン | プロジェクト編集画面 |
| プロジェクト詳細画面 | WBSボタン | WBS一覧画面 |
| プロジェクト詳細画面 | ガントチャートボタン | ガントチャート画面 |
| プロジェクト詳細画面 | ロードマップボタン | ロードマップ画面 |
| プロジェクト詳細画面 | ダッシュボードボタン | プロジェクト別ダッシュボード画面 |

なし

## 8. アクション

### 画面表示時

1. `GET /api/v1/projects/{id}/` 呼び出し
2. プロジェクト情報をストアに格納して表示

### 削除確認「削除」押下時

1. `DELETE /api/v1/projects/{id}/` 呼び出し
2. 成功（204）: `/projects` へ遷移
