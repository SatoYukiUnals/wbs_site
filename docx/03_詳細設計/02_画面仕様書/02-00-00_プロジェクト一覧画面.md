# 【画面仕様書】プロジェクト一覧画面

## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 02-00-00 |
| 画面名 | プロジェクト一覧画面 |
| URL | `/projects` |
| 対応デザイン | - |

## 2. 目的

ログイン中ユーザーが参加しているプロジェクトの一覧を表示し、各プロジェクトへのナビゲーションを提供する。

## 3. 処理概要

### 画面で操作できる情報

- 参加プロジェクト一覧の表示
- プロジェクト詳細画面への遷移
- プロジェクト作成画面への遷移（admin以上）

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| ダッシュボード | プロジェクト一覧リンク | プロジェクト一覧画面 |
| ログイン | ログイン後 | ダッシュボード（または前回アクセス先） |
| プロジェクト一覧画面 | プロジェクト行クリック | プロジェクト詳細画面 |
| プロジェクト一覧画面 | 新規作成ボタン | プロジェクト作成画面 |

## 4. アクセス権限

| ロール | アクセス可否 |
| :--- | :--- |
| 全ユーザー（ログイン済み） | ○ |
| 未ログイン | - |

## 8. アクション

### 画面表示時

1. `GET /api/v1/projects/` 呼び出し
2. プロジェクト一覧をストアに格納して表示
