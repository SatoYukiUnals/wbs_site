# 【画面仕様書】テンプレート一覧画面

## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 11-00-00 |
| 画面名 | テンプレート一覧画面 |
| URL | `/templates` |
| 対応デザイン | - |

## 2. 目的

自分が作成した個人テンプレートと共有テンプレートの一覧を表示し、テンプレートの管理を行う。

## 3. 処理概要

### 画面で操作できる情報

- テンプレート一覧の表示（種別フィルタ付き）
- テンプレートの作成・編集・削除

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| ヘッダーメニュー | テンプレートリンク | テンプレート一覧画面 |
| テンプレート一覧画面 | 新規作成ボタン | テンプレート作成・編集画面 |
| テンプレート一覧画面 | 編集ボタン | テンプレート作成・編集画面 |

## 7. バリデーション

なし

## 8. アクション

### 画面表示時

1. `GET /api/v1/templates/` 呼び出し
2. テンプレート一覧を表示

### 種別フィルタ変更時

1. `GET /api/v1/templates/?type={type}` 呼び出し

### 削除確認「削除」押下時

1. `DELETE /api/v1/templates/{template_id}/` 呼び出し
2. 成功（204）: 一覧を再取得して表示
