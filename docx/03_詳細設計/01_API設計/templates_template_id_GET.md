# 【API設計書】テンプレート詳細取得

## 1. 概要

指定テンプレートの詳細情報（タイトル・種別・内容）を返す。自分が作成したテンプレートまたは共有テンプレートのみアクセス可能。

## 2. エンドポイント

- **Method:** `GET`
- **Path:** `/api/v1/templates/{template_id}/`
- **認証:** JWT必須（全ユーザー）

## 3. リクエスト

### パスパラメータ

#### `template_id`（テンプレートID）

- **型:** `integer` / **必須:** ○

## 4. レスポンス

### 成功 (200 OK)

```json
{
  "id": 1,
  "title": "標準開発フロー",
  "type": "wbs",
  "content": "フロントエンド実装\nバックエンド実装\nテスト\nレビュー依頼",
  "is_shared": true,
  "created_by": {
    "user_id": 1,
    "username": "hanako"
  }
}
```

### エラー (4xx / 5xx)

| 状況 | ステータス | エラーコード | メッセージ内容 | 備考 |
| :--- | :--- | :--- | :--- | :--- |
| 未認証 | 401 | `ERR_AUTH_002` | 認証が必要です | |
| 対象未存在・アクセス不可 | 404 | `ERR_DAT_404` | 対象のデータが見つかりませんでした | 他人の個人テンプレートも404 |
| サーバーエラー | 500 | `ERR_SYS_500` | システムエラーが発生しました | |

## 5. 内部処理 / ロジック

1. **JWT検証**
   - `user_id`・`tenant_id` を取得
     - 無効：401 `ERR_AUTH_002` を返す
2. **データ取得・アクセス確認**
   - `Template` を `template_id` で SELECT
   - `(is_shared = True AND tenant_id = JWTのtenant_id) OR created_by = user_id` の条件を満たすことを確認
     - 未存在 or アクセス不可：404 `ERR_DAT_404` を返す
3. **レスポンス生成**
   - テンプレート詳細を返却
