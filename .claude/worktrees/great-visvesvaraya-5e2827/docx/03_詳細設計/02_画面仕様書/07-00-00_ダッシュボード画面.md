## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |

| URL | `/dashboard` |
| 対応デザイン | - |

## 2. 目的

ログイン中ユーザーが参加する全プロジェクトのサマリー情報を一覧表示する。

## 3. 処理概要

### 画面で操作できる情報

- 参加プロジェクトの進捗サマリー表示
- 自分が担当するタスクの一覧（期限切れ・直近）
- 各プロジェクトへのクイックアクセス

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| ログイン | ログイン成功 | ダッシュボード画面 |
| ヘッダー | ダッシュボードリンク | ダッシュボード画面 |
| ダッシュボード画面 | プロジェクトカードクリック | プロジェクト詳細画面 |
| ダッシュボード画面 | タスク名クリック | タスク詳細画面 |
