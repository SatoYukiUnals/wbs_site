# 02-01-03 プロジェクト詳細画面

## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | 02-01-03 |
| 画面名 | プロジェクト詳細画面 |
| URL | `/projects/{project_id}` |

---

## 目的

プロジェクト概要・進捗率を表示し、各機能（WBS・ガントチャート・ロードマップ等）へのナビゲーションを提供する。

---

## 処理概要

### ■ 画面の遷移

| 種別 | 画面 |
|------|------|
| 遷移元 | 02-01-00 プロジェクト一覧 |
| 遷移先 | 02-01-02 編集 / 02-02-00 メンバー管理 / 03-01-00 クォーター / 04-01-00 WBS / 05-01-00 ガント / 06-01-00 ロードマップ / 08-01-00 レビュー / 09-01-04 報告書 / 10-01-04 Excel |

---

## アクセス権限

| 種別 | 内容 |
|------|------|
| アクセス権限 | プロジェクトメンバー以上 |

---

## ダイアログ

- 削除確認ダイアログ：「このプロジェクトを削除しますか？」

---

## 機能仕様

### ■ 出力フィールド

| 項目名 | 内容 | 初期値 |
|--------|------|--------|
| プロジェクト名 | プロジェクト名を表示 | APIから取得 |
| 説明 | 説明を表示 | APIから取得 |
| 期間 | 開始日〜終了日を表示 | APIから取得 |
| 進捗率 | 数値（%）とプログレスバーで表示 | APIから取得 |

### ■ ボタン

| ボタン名 | 挙動 |
|---------|------|
| 編集（admin以上） | 遷移先: `02-01-02` |
| 削除（admin以上） | 確認ダイアログ → `DELETE /api/v1/projects/{project_id}/` |
| メンバー管理 | 遷移先: `02-02-00` |
| WBS | 遷移先: `04-01-00` |
| ガントチャート | 遷移先: `05-01-00` |
| ロードマップ | 遷移先: `06-01-00` |
| クォーター管理 | 遷移先: `03-01-00` |
| レビュー | 遷移先: `08-01-00` |
| 報告書 | 遷移先: `09-01-04` |
| Excel出力 | 遷移先: `10-01-04` |

---

## アクション

### ■ 画面遷移時

- API: `GET /api/v1/projects/{project_id}/`
