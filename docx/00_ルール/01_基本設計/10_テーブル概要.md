# 機能仕様書：テーブル概要

## 1. ■概要・目的

対象機能で操作するテーブルとカラムを明確にします。**テーブルごとに小項目（H3 `### ■ {テーブル名}`）として切り出し、テーブル単位の表で記述**します。テーブル定義との整合性を確保します。

## 2. ■事前準備

- [ ] 処理フロー（05）・処理ロジック詳細（06）の記述完了
- [ ] 参照手順書：[テーブル定義手順書](../05_テーブル定義.md)

## 3. ■作業手順（Issueのタスクリストとして利用）

- [ ] **対象テーブルの特定**：処理フローで操作するテーブルをすべてリストアップします。
- [ ] **テーブル単位の小項目作成**：各テーブルを `### ■ {テーブル名}（{論理名}）` の小項目として切り出します。
- [ ] **カラムの列挙**：各テーブルの小項目の下に、操作対象のカラムを1行1カラム形式で表にまとめます。
- [ ] **操作種別の記載**：各カラムへの操作（INSERT / SELECT / UPDATE / DELETE）を記載します。

## 4. ■セルフチェック項目（作業完了後の最終確認）

- [ ] 処理フロー（05）で操作するテーブル・カラムがすべて記載されている
- [ ] **テーブルごとに小項目（H3）が分かれており、テーブル単位の表になっている**
- [ ] 1行1カラム形式で記述されている
- [ ] 操作種別（INSERT / SELECT / UPDATE / DELETE）が明記されている
