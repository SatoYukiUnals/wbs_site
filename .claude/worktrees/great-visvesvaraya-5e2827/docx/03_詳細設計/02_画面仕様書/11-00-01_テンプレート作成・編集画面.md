# 【画面仕様書】テンプレート作成・編集画面

## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 11-00-01 |
| 画面名 | テンプレート作成・編集画面 |
| URL | `/templates/new`（作成） / `/templates/{id}/edit`（編集） |
| 対応デザイン | - |

## 2. 目的

テンプレートの新規作成または編集を行う。全ユーザーが個人テンプレートを作成可能。共有設定（is_shared=true）はadmin以上のみ可能。

## 3. 処理概要

### 画面で操作できる情報

- テンプレートタイトル・種別・内容・共有設定の入力
- テンプレートの保存

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| テンプレート一覧画面 | 新規作成・編集ボタン | テンプレート作成・編集画面 |
| テンプレート作成・編集画面 | 保存成功 | テンプレート一覧画面 |
| テンプレート作成・編集画面 | キャンセルボタン | テンプレート一覧画面 |

## 4. アクセス権限

## 7. バリデーション

| 項目名 | トリガー | チェック内容 | エラーID | エラーメッセージ |
| :--- | :--- | :--- | :--- | :--- |
| タイトル | submit | 必須・1〜200文字 | ERR_VAL_001 | タイトルを入力してください |
| 種別 | submit | wbs / task のいずれか | ERR_VAL_001 | 種別を選択してください |
| 内容 | submit | 必須・1行以上 | ERR_VAL_001 | 内容を1行以上入力してください |
| 共有設定 | submit | is_shared=true時はadmin以上のみ | ERR_PERM_001 | 共有設定を変更する権限がありません |

## 8. アクション

### 画面表示時（編集モード）

1. `GET /api/v1/templates/{id}/` 呼び出し
2. 取得データをフォームに設定

### 保存ボタン押下時

1. フロント側バリデーション実行
2. API呼び出し（POST or PUT）
3. 成功: `/templates` へ遷移
