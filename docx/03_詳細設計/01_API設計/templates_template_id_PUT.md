# 【API設計書】テンプレート編集

## 1. 概要

指定テンプレートのタイトル・種別・内容・共有設定を更新する。作成者本人または admin 以上のみ実行可能。`is_shared = True` への変更は admin 以上のみ可能。

## 2. エンドポイント

- **Method:** `PUT`
- **Path:** `/api/v1/templates/{template_id}/`
- **認証:** JWT必須（作成者本人 or admin 以上）

## 3. リクエスト

### パスパラメータ

#### `template_id`（テンプレートID）

- **型:** `integer` / **必須:** ○

### ボディ

#### `title`（テンプレートタイトル）

- **型:** `string` / **必須:** ○
- 1〜200文字

#### `type`（種別）

- **型:** `string` / **必須:** ○
- `wbs` または `task` のいずれか

#### `content`（テンプレート内容）

- **型:** `string` / **必須:** ○
- 改行区切りのテキスト。1行以上

#### `is_shared`（共有設定）

- **型:** `boolean` / **必須:** -
- `true` への変更は admin 以上のみ

## 4. レスポンス

### 成功 (200 OK)

```json
{
  "id": 1,
  "title": "標準開発フロー 更新版",
  "type": "wbs",
  "content": "要件確認\nフロントエンド実装\nバックエンド実装\nテスト\nレビュー依頼",
  "is_shared": true
}
```

### エラー (4xx / 5xx)

| 状況 | ステータス | エラーコード | メッセージ内容 | 備考 |
| :--- | :--- | :--- | :--- | :--- |
| 未認証 | 401 | `ERR_AUTH_002` | 認証が必要です | |
| 権限不足 | 403 | `ERR_PERM_001` | この操作を行う権限がありません | 作成者でも非admin |
| バリデーションエラー | 400 | `ERR_VAL_001` | 入力内容に不備があります | |
| 対象未存在 | 404 | `ERR_DAT_404` | 対象のデータが見つかりませんでした | |
| サーバーエラー | 500 | `ERR_SYS_500` | システムエラーが発生しました | |

## 5. 内部処理 / ロジック

1. **JWT検証**
   - `user_id`・`role`・`tenant_id` を取得
     - 無効：401 `ERR_AUTH_002` を返す
2. **対象テンプレート確認**
   - `Template` を `template_id` で SELECT（同テナントまたは自分が作成）
     - 未存在：404 `ERR_DAT_404` を返す
3. **権限チェック**
   - `created_by = user_id` または `role` が `admin` 以上であることを確認
     - 権限外：403 `ERR_PERM_001` を返す
4. **共有設定変更の権限チェック**（`is_shared = True` に変更する場合のみ）
   - `role` が `admin` または `master` であることを確認
     - 権限不足：403 `ERR_PERM_001` を返す
5. **バリデーション**
   - `content` が1行以上であることを確認
     - エラー時：400 `ERR_VAL_001` を返す
6. **DB更新**
   - `Template` レコードを UPDATE
     - DBエラー：500 `ERR_SYS_500` を返す
7. **レスポンス生成**
   - 更新後のテンプレート情報を返却
