# 【画面仕様書】Excel出力設定画面

## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 10-00-04 |
| 画面名 | Excel出力設定画面 |
| URL | `/projects/{id}/export/excel` |
| 対応デザイン | - |

## 2. 目的

プロジェクトのタスク一覧とガントチャートをExcel（.xlsx）形式でエクスポートする。クォーターを指定して絞り込み出力が可能。

## 3. 処理概要

### 画面で操作できる情報

- 出力対象のクォーター選択
- Excelファイルのダウンロード

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| プロジェクト詳細画面 | Excel出力リンク | Excel出力設定画面 |
| Excel出力設定画面 | 戻るボタン | プロジェクト詳細画面 |

| :--- | :--- |

なし

## 8. アクション

### 画面表示時

1. `GET /api/v1/projects/{id}/quarters/` でクォーター選択肢を取得

### ダウンロードボタン押下時

1. `GET /api/v1/projects/{id}/export/excel/` （クォーター選択時: `?quarter_id={id}`） 呼び出し
2. .xlsxファイルのダウンロードを開始

## 9. 状態管理

| isLoading | boolean | false | ダウンロード中フラグ |
