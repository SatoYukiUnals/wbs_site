# 【画面仕様書】レビュー実施画面

## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 08-00-02 |
| 画面名 | レビュー実施画面 |
| URL | `/projects/{id}/tasks/{task_id}/review` |
| 対応デザイン | - |

## 2. 目的

タスクのレビューを実施し、承認または差し戻しを行う。担当者以外（非担当者のメンバー）のみ操作可能。

## 3. 処理概要

### 画面で操作できる情報

- タスク詳細・レビュー申請内容の参照
- コメントの入力
- 承認または差し戻しの実行

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| レビュー一覧画面 | レビューボタン | レビュー実施画面 |
| タスク詳細画面 | レビューボタン | レビュー実施画面 |
| レビュー実施画面 | 承認・差し戻し成功 | レビュー一覧画面 |
| レビュー実施画面 | キャンセルボタン | レビュー一覧画面 |

## 8. アクション

### 画面表示時

1. `GET /api/v1/projects/{id}/tasks/{task_id}/` でタスク情報を取得
2. `GET /api/v1/projects/{id}/tasks/{task_id}/reviews/` でレビュー情報を取得

### 承認ボタン押下時

1. `POST /api/v1/projects/{id}/tasks/{task_id}/reviews/approve/` 呼び出し
2. 成功（200）: `/projects/{id}/reviews` へ遷移

### 差し戻しボタン押下時

1. コメント必須チェック
2. `POST /api/v1/projects/{id}/tasks/{task_id}/reviews/reject/` 呼び出し
3. 成功（200）: `/projects/{id}/reviews` へ遷移
