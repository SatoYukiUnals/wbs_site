# 【画面仕様書】自動割り振りプレビュー画面

## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 04-02-04 |
| 画面名 | 自動割り振りプレビュー画面 |
| URL | `/projects/{id}/auto-assign/preview` |
| 対応デザイン | - |

## 2. 目的

担当者自動割り当てのシミュレーション結果をプレビューし、確認・確定を行う。

## 3. 処理概要

### 画面で操作できる情報

- 自動割り振りのプレビュー結果表示
- 割り振り結果の確定または破棄

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| WBS一覧画面 | 自動割り振りボタン | 自動割り振りプレビュー画面 |
| 自動割り振りプレビュー画面 | 確定ボタン | WBS一覧画面 |
| 自動割り振りプレビュー画面 | キャンセルボタン | WBS一覧画面 |

## 7. バリデーション

なし

## 8. アクション

### 画面表示時

1. `POST /api/v1/projects/{id}/auto-assign/preview/` 呼び出し
2. プレビュー結果を表示

### 確定確認ダイアログ「確定」押下時

1. `POST /api/v1/projects/{id}/auto-assign/confirm/` 呼び出し
2. 成功（200）: `/projects/{id}/wbs` へ遷移し成功トーストを表示
