# 【画面仕様書】タスク詳細画面

## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 04-01-03 |
| 画面名 | タスク詳細画面 |
| URL | `/projects/{id}/tasks/{task_id}` |
| 対応デザイン | - |

## 2. 目的

タスクの詳細情報（名称・説明・期間・ステータス・進捗率・担当者・レビュー状態）を参照する。

## 3. 処理概要

### 画面で操作できる情報

- タスク詳細情報の参照
- タスク編集画面への遷移（admin以上または担当者）
- レビュー申請・実施への遷移

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| WBS一覧画面 | タスク名クリック | タスク詳細画面 |
| タスク詳細画面 | 編集ボタン | タスク編集画面 |
| タスク詳細画面 | レビューボタン | レビュー実施画面 |

| :--- | :--- |
