# 【画面仕様書】ロードマップアイテム作成・編集画面

## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 06-01-01 |
| 画面名 | ロードマップアイテム作成・編集画面 |
| URL | `/projects/{id}/roadmap/new`（作成） / `/projects/{id}/roadmap/{item_id}/edit`（編集） |
| 対応デザイン | - |

## 2. 目的

ロードマップアイテム（マイルストーン・フェーズ）の新規作成または編集を行う。admin以上のみ操作可能。

## 3. 処理概要

### 画面で操作できる情報

- アイテム名・種別・開始日・終了日・説明の入力
- アイテムの保存

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| ロードマップ画面 | 新規作成・編集ボタン | ロードマップアイテム作成・編集画面 |
| ロードマップアイテム作成・編集画面 | 保存成功 | ロードマップ画面 |
| ロードマップアイテム作成・編集画面 | キャンセルボタン | ロードマップ画面 |

## 4. アクセス権限

## 7. バリデーション

| 項目名 | トリガー | チェック内容 | エラーID | エラーメッセージ |
| :--- | :--- | :--- | :--- | :--- |
| アイテム名 | submit | 必須・1〜200文字 | ERR_VAL_001 | アイテム名を入力してください |
| 種別 | submit | milestone / phase のいずれか | ERR_VAL_001 | 種別を選択してください |
| 開始日 | submit | 必須・日付形式 | ERR_VAL_001 | 開始日を入力してください |
| 終了日 | submit | 開始日以降（入力時） | ERR_VAL_001 | 終了日は開始日以降の日付を入力してください |

## 8. アクション

### 画面表示時（編集モード）

1. `GET /api/v1/projects/{id}/roadmap/{item_id}/` 呼び出し
2. 取得データをフォームに設定

### 保存ボタン押下時

1. フロント側バリデーション実行
2. API呼び出し（POST or PUT）
3. 成功: `/projects/{id}/roadmap` へ遷移
