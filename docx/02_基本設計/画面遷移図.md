# 画面遷移図

- **最終更新日**：2026-04-26
- **バージョン**：v1.0

---

## 1. 全体遷移概要

ログイン後はダッシュボード（07-01-00）が起点となる。プロジェクトを選択すると、そのプロジェクト内の各機能画面へ遷移する。

---

## 2. 認証フロー（01）

```mermaid
flowchart TD
    Start([アクセス]) --> Auth{ログイン済み?}
    Auth -->|Yes| Dashboard["07-01-00\nダッシュボード"]
    Auth -->|No| Login["01-01-04\nログイン画面"]
    Login -->|ログイン成功| Dashboard
    Login -->|テナント未作成| TenantReg["01-01-01\nテナント登録画面"]
    TenantReg -->|登録完了| Dashboard
    Dashboard --> Profile["01-03-03\nプロフィール画面"]
    Profile --> Dashboard
```

---

## 3. ユーザー管理フロー（01）

```mermaid
flowchart TD
    Dashboard["07-01-00\nダッシュボード"] --> UserList["01-02-00\nユーザー管理画面"]
    UserList --> InviteUser["01-02-05\nユーザー招待画面"]
    InviteUser -->|招待完了| UserList
    UserList -->|ロール変更・削除| UserList
```

---

## 4. プロジェクト管理フロー（02）

```mermaid
flowchart TD
    Dashboard["07-01-00\nダッシュボード"] --> PjList["02-01-00\nプロジェクト一覧"]
    PjList --> PjNew["02-01-01\nプロジェクト登録"]
    PjNew -->|登録完了| PjDetail["02-01-03\nプロジェクト詳細"]
    PjList --> PjDetail
    PjDetail --> PjEdit["02-01-02\nプロジェクト編集"]
    PjEdit -->|保存| PjDetail
    PjDetail --> MemberList["02-02-00\nメンバー管理"]
    MemberList --> PjDetail
```

---

## 5. プロジェクト内フロー（03〜11）

プロジェクト詳細（02-01-03）からプロジェクト内の各機能へ遷移する。

```mermaid
flowchart TD
    PjDetail["02-01-03\nプロジェクト詳細"] --> QuarterList["03-01-00\nクォーター管理"]
    PjDetail --> WBS["04-01-00\nWBS一覧"]
    PjDetail --> Gantt["05-01-00\nガントチャート"]
    PjDetail --> Roadmap["06-01-00\nロードマップ"]
    PjDetail --> ReviewList["08-01-00\nレビュー一覧"]
    PjDetail --> Report["09-01-04\n報告書作成・編集"]
    PjDetail --> Excel["10-01-04\nExcel出力設定"]

    QuarterList --> QuarterNew["03-01-01\nクォーター登録"]
    QuarterNew -->|登録完了| QuarterList
    QuarterList --> QuarterEdit["03-01-02\nクォーター編集"]
    QuarterEdit -->|保存| QuarterList
```
