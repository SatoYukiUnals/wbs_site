# 【画面仕様書】ガントチャート画面

## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 05-00-00 |
| 画面名 | ガントチャート画面 |
| URL | `/projects/{id}/gantt` |
| 対応デザイン | - |

## 2. 目的

プロジェクトのタスクをガントチャート形式で時系列表示し、スケジュールを可視化する。ガントチャートは自作（Canvas / SVG）。

## 3. 処理概要

### 画面で操作できる情報

- タスクのガントチャート表示（バー形式）
- 表示期間の切り替え（日・週・月単位）
- クォーターフィルタ

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| プロジェクト詳細画面 | ガントチャートリンク | ガントチャート画面 |
| ガントチャート画面 | タスク名クリック | タスク詳細画面 |

## 8. アクション

### 画面表示時

1. `GET /api/v1/projects/{id}/gantt/` 呼び出し
2. `GET /api/v1/projects/{id}/quarters/` でクォーター選択肢を取得
3. ガントチャートをCanvas/SVGで描画

### フィルタ変更時

1. `GET /api/v1/projects/{id}/gantt/?quarter_id={id}` 呼び出し
2. チャートを再描画
