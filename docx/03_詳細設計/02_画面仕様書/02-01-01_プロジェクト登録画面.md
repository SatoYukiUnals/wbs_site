# 02-01-01 プロジェクト登録画面

## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | 02-01-01 |
| 画面名 | プロジェクト登録画面 |
| URL | `/projects/new` |

---

## 目的

プロジェクトの名前・説明・期間を入力して新規作成する。

---

## 処理概要

### ■ 画面の遷移

| 種別 | 画面 |
|------|------|
| 遷移元 | 02-01-00 プロジェクト一覧 |
| 遷移先（登録成功） | 02-01-03 プロジェクト詳細 |

---

## アクセス権限

| 種別 | 内容 |
|------|------|
| アクセス権限 | admin 以上 |

---

## 機能仕様

### ■ 入力フィールド

| 項目名 | 必須 | 型 | 初期値 | データ型 |
|--------|------|----|--------|---------|
| プロジェクト名 | ✅ | テキストボックス | - | string |
| 説明 | - | テキストエリア | - | string |
| 開始日 | ✅ | 日付 | - | date |
| 終了日 | ✅ | 日付 | - | date |

### ■ ボタン

| ボタン名 | 挙動 |
|---------|------|
| 登録 | `POST /api/v1/projects/` を呼び出す |
| キャンセル | プロジェクト一覧へ戻る |

---

## バリデーション

### ■ 終了日

- **トリガー:** 登録ボタン押下時

| チェック | 追加ルール | エラーID | エラーメッセージ |
|---------|-----------|---------|----------------|
| 開始日より後 | - | - | 終了日は開始日より後に設定してください |

---

## 更新テーブル

### ■ `project`

| 項目名 | カラム | データ型 |
|--------|--------|---------|
| プロジェクト名 | name | VARCHAR |
| 説明 | description | TEXT |
| 開始日 | start_date | DATE |
| 終了日 | end_date | DATE |
