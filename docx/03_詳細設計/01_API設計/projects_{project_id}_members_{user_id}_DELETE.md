# 【API設計書】メンバー削除API

## 1. 概要

プロジェクトからメンバーを除外する。

## 2. エンドポイント

- **Method:** `DELETE`
- **Path:** `/api/v1/projects/{project_id}/members/{user_id}/`
- **認証:** 必要（Bearer トークン）
- **権限:** プロジェクト内 admin 以上

## 3. リクエスト

### パスパラメータ

| パラメータ | 型 | 必須 | 説明 |
|---|---|---|---|
| `project_id` | UUID | ✅ | 対象プロジェクトID |
| `user_id` | UUID | ✅ | 削除対象ユーザーID |

## 4. レスポンス

### 成功 (204 No Content)

ボディなし

### エラー (4xx)

| 状況 | ステータス | メッセージ内容 |
|---|---|---|
| 対象メンバーが存在しない | 404 | 対象メンバーが見つかりません |
| 権限不足 | 403 | この操作を行う権限がありません |

## 5. 内部処理 / ロジック

1. **権限チェック** → admin 以上か確認
2. **メンバー検索** → `project_member` テーブルで `project_id` + `user_id` 検索。なければ 404
3. **DB削除** → `project_member` レコードを DELETE
4. **レスポンス** → 204 No Content を返す
