**作成日：** 2026年4月12日

## 画面ID 体系

```
[カテゴリ番号]-[データ単位番号]-[操作区分]

カテゴリ番号   : カテゴリごとの固定番号（01〜11）
データ単位番号 : カテゴリ内のデータ単位ごとに採番（同じデータは同じ番号）
操作区分       : 00=一覧 / 01=登録 / 02=変更 / 03=詳細 / 04以降=その他

カテゴリ番号一覧:
  01 : 認証・ユーザー管理
  02 : プロジェクト管理
  03 : クォーター管理
  04 : WBS・タスク管理
  05 : ガントチャート
  06 : プロダクトロードマップ
  07 : ダッシュボード
  08 : レビュー管理
  09 : 報告書管理
  10 : Excel出力
  11 : テンプレート管理
```
