# 【画面仕様書】WBS一覧画面

## 1. 基本情報

| 項目 | 内容 |
| :--- | :--- |
| 画面ID | 04-00-00 |
| 画面名 | WBS一覧画面 |
| URL | `/projects/{id}/wbs` |
| 対応デザイン | - |

## 2. 目的

プロジェクトのタスクをWBS（Work Breakdown Structure）形式でツリー表示し、タスクの管理を行う。

## 3. 処理概要

### 画面で操作できる情報

- タスクの階層ツリー表示（最大5階層）
- タスクの作成・編集・削除（admin以上）
- タスクの並び替え（ドラッグ＆ドロップ）
- テンプレートからの一括作成
- 担当者割り当て

### 画面の遷移

| 遷移元 | 操作 | 遷移先 |
| :--- | :--- | :--- |
| プロジェクト詳細画面 | WBSリンク | WBS一覧画面 |
| WBS一覧画面 | タスク名クリック | タスク詳細画面 |
| WBS一覧画面 | 新規作成ボタン | タスク作成画面 |
| WBS一覧画面 | 一括作成ボタン | タスク一括作成画面 |
| WBS一覧画面 | 自動割り振りボタン | 自動割り振りプレビュー画面 |

## 7. バリデーション

なし

## 8. アクション

### 画面表示時

1. `GET /api/v1/projects/{id}/tasks/` 呼び出し
2. タスクをツリー構造で表示

### ドラッグ＆ドロップ後

1. `PATCH /api/v1/projects/{id}/tasks/{task_id}/order/` 呼び出し
2. 表示を更新

### 削除確認「削除」押下時

1. `DELETE /api/v1/projects/{id}/tasks/{task_id}/` 呼び出し
2. 成功（204）: 一覧を再取得して表示
