# 【API設計書】テンプレート作成

## 1. 概要

テンプレートを新規作成する。`is_shared = True`（共有設定）は admin 以上のみ可能。全ユーザーが個人テンプレートを作成できる。

## 2. エンドポイント

- **Method:** `POST`
- **Path:** `/api/v1/templates/`
- **認証:** JWT必須（全ユーザー）

## 3. リクエスト

### ボディ

#### `title`（テンプレートタイトル）

- **型:** `string` / **必須:** ○
- 1〜200文字

#### `type`（種別）

- **型:** `string` / **必須:** ○
- `wbs` または `task` のいずれか

#### `content`（テンプレート内容）

- **型:** `string` / **必須:** ○
- 改行区切りのテキスト。1行以上

#### `is_shared`（共有設定）

- **型:** `boolean` / **必須:** -
- `true` の場合は admin 以上のみ指定可能。省略時は `false`

## 4. レスポンス

### 成功 (201 Created)

```json
{
  "id": 3,
  "title": "標準開発フロー",
  "type": "wbs",
  "content": "フロントエンド実装\nバックエンド実装\nテスト\nレビュー依頼",
  "is_shared": false,
  "created_by": 1
}
```

### エラー (4xx / 5xx)

| 状況 | ステータス | エラーコード | メッセージ内容 | 備考 |
| :--- | :--- | :--- | :--- | :--- |
| 未認証 | 401 | `ERR_AUTH_002` | 認証が必要です | |
| 共有設定の権限不足 | 403 | `ERR_PERM_001` | この操作を行う権限がありません | is_shared=Trueのとき |
| バリデーションエラー | 400 | `ERR_VAL_001` | 入力内容に不備があります | |
| サーバーエラー | 500 | `ERR_SYS_500` | システムエラーが発生しました | |

## 5. 内部処理 / ロジック

1. **JWT検証**
   - `user_id`・`role`・`tenant_id` を取得
     - 無効：401 `ERR_AUTH_002` を返す
2. **バリデーション**
   - `title`・`type`・`content` の存在チェック
   - `type` が `wbs` または `task` であることを確認
   - `content` が1行以上であることを確認（`\n` 分割後に要素が存在すること）
     - エラー時：400 `ERR_VAL_001` を返す
3. **共有設定権限チェック**（`is_shared = True` の場合のみ）
   - `role` が `admin` または `master` であることを確認
     - 権限不足：403 `ERR_PERM_001` を返す
4. **DB操作**
   - `Template` レコードを INSERT（`created_by` = `user_id`・`tenant_id` = JWTの `tenant_id`）
     - DBエラー：500 `ERR_SYS_500` を返す
5. **レスポンス生成**
   - 作成したテンプレート情報を返却
